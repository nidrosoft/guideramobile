## DESIGN SYSTEM COLORS

| Token | Value | Usage |
|-------|-------|-------|
| **Primary (Green)** | `#3FC39E` | Buttons, accents, active states, highlights |
| **Background** | `#202020` | App background (dark mode) |
| **Card Background** | `#1A1A1A` | Card/elevated surfaces |
| **Text Primary** | `rgba(248,248,248,0.95)` | Main text |
| **Text Secondary** | `rgba(248,248,248,0.6)` | Subtitles, labels |
| **Border** | `rgba(248,248,248,0.08)` | Card borders, dividers |

The app is **dark-mode-first** — all screenshots should reflect the dark theme with green accents.
