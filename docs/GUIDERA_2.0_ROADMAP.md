# GUIDERA 2.0 — Feature Roadmap

**Date:** March 19, 2026
**Source:** Competitive audit of 10+ travel apps (TripIt, Expedia, Wanderlog, Airbnb, Google Travel, Hopper, etc.)
**Methodology:** Distilled top 100 recurring complaints and missing features into actionable items, cross-referenced with Guidera's current capabilities.

---

## How This Document Works

Each feature is tagged with:
- **Status**: ✅ Already Built | 🟡 Partially Built | ❌ Not Built
- **Effort**: 🟢 Easy (< 1 day) | 🟡 Medium (1-3 days) | 🔴 Hard (1+ week)
- **Priority**: P0 (launch blocker) | P1 (first sprint) | P2 (second sprint) | P3 (future)
- **Category**: Which part of the app it lives in

Items are organized from easiest to most complex within each category.
